# 第十三讲：开放、后发优势与中美关系的转折

改革与开放不能分开理解。对内改革释放劳动力、地方积极性和市场空间；对外开放则把中国重新接入世界资本、技术、市场和知识网络。没有改革，开放会缺少内部承接能力；没有开放，改革也容易困在国内资源和技术边界中。本讲的关键词是“后发优势”：后来者可以学习、引进、模仿和赶超，但也可能因为过度依赖外部经验而形成新的脆弱。

## 一、为什么必须开放

近代中国曾长期在被迫开放中承受屈辱，因此“开放”这个词在历史上并不轻松。改革开放时期的开放，关键区别在于主动性：不是在炮舰压力下割让主权，而是在国家主导下选择性引进资本、技术、管理和市场联系。

邓小平反复强调不能关起门来搞建设，其现实背景很清楚：经历长期计划体制和“文革”破坏后，中国技术落后、资金短缺、管理经验不足，而世界经济正在发生产业转移。若继续封闭，国内改革难以获得外部资源；若开放得当，后发国家可以节省试错成本。

## 二、后发优势的含义

后发优势指相对落后的国家在现代化过程中，可以直接学习先发国家已有技术、制度经验和产业组织方式，从而以较低成本实现追赶。它包含三个层面：技术上可引进设备和工艺，制度上可借鉴管理与法律经验，产业上可承接先发国家转移出的劳动密集型制造环节。

中国改革开放初期正好具备承接条件：劳动力充足，基础教育已有一定积累，国家能提供稳定秩序，港澳台和海外华人资本拥有语言、亲缘和地缘优势。珠江三角洲最早起飞，并非偶然；它离香港近，地方政府灵活，乡镇和村社能迅速组织土地、劳动力和厂房。

## 三、经济特区的试验意义

深圳、珠海、汕头、厦门等经济特区，是开放政策的制度试验田。特区不是简单给优惠，而是允许在土地使用、外资引进、劳动用工、企业管理、进出口制度上先行突破。它们的真正价值，在于把抽象争论转化为可观察结果：如果试点有效，再逐步推广；如果出现问题，也能把风险限制在局部。

这种方法与农村改革相似，都是“先试、再看、再扩”。中国改革开放的一个重要经验，正是用空间上的局部试验来处理意识形态和制度不确定性。它不是一次性设计好的蓝图，而是在试错中不断扩大可行边界。

## 四、中美关系与开放窗口

1970年代中美关系破冰，为对外开放提供了重要国际条件。1972年尼克松访华和《上海公报》，首先是冷战格局中的战略调整；1979年中美建交，则使经贸、科技、教育交流获得制度基础。邓小平访美带来的象征意义很强：中国向世界表达了学习和合作的意愿，也向国内释放开放信号。

1980年代中美关系并非没有摩擦，台湾问题、意识形态差异和战略互疑一直存在。但在共同制衡苏联的大背景下，双方形成较大合作空间。中国派出留学生，引进设备和管理经验，参与国际金融与贸易联系，开放由此不再只是沿海招商，而逐渐成为国家发展战略的一部分。

## 五、开放的社会后果

开放改变的不只是经济数字。它带来新的消费品、新的职业想象、新的城市空间和新的知识结构。留学潮、外企、合资企业、沿海加工贸易、个体户与私营经济，共同塑造了一代人的生活经验。人们开始用收入、效率、市场、竞争、产权、合同等词汇重新理解社会运行。

同时，开放也带来不平衡。沿海与内地差距扩大，外向型地区更快融入全球产业链，内陆地区则相对滞后；市场机会增加，也伴随腐败、走私、收入差距和价值观冲突。开放不是纯粹的红利，而是把国内社会带入更复杂的世界竞争。

## 六、后发劣势与模仿陷阱

后发优势若处理不好，会转化为后发劣势。最常见的陷阱，是只模仿技术和管理表层，而不建立自主创新、法治秩序和制度韧性。晚清洋务运动已经提供过教训：可以买机器、造船厂、办学堂，但如果财政、官僚、教育、军队和政治责任机制不变，器物现代化很难支撑国家现代化。

改革开放后的中国也面对类似问题。加工贸易能快速积累外汇和就业，却可能被锁定在低附加值环节；引进技术能缩短差距，却可能抑制原创能力；地方招商能推动增长，也可能带来环境代价和债务冲动。因此开放的下一步，一定是从“引进来”走向“消化、吸收、再创新”。

## 七、民粹主义与全球化退潮

开放依赖相对稳定的全球化环境。但全球化并不会让所有国家、所有阶层平均受益。产业转移使部分发达国家工人感到被抛下，移民、贸易逆差、金融危机又刺激民粹主义上升。进入21世纪后，全球化退潮、贸易保护和地缘竞争加剧，中国的发展外部环境明显变化。

这使改革开放的历史经验需要更新：早期可以更多依靠低成本、外资、出口和模仿；后期则必须面对技术封锁、供应链安全、国内消费、人口老龄化和创新能力。开放仍然必要，但开放的方式必须更成熟。

## 八、本讲结论

开放的本质，是把国内改革放进世界坐标中检验和推进。它给中国带来后发优势，也带来依赖、竞争和不平衡。理解开放，既不能忘记近代被迫开放的屈辱，也不能把主动开放简单理解成向外部让步。真正的问题是：一个后发大国如何在学习世界的同时，逐步形成自己的制度能力、产业能力和知识能力。

## 九、从“引进来”到“走出去”

开放早期的重点是引进资金、设备、技术和管理经验。合资企业、加工贸易、外商投资和留学生归国，把外部知识不断带入中国。随着产业能力提升，中国企业开始走出去，参与海外工程、资源合作、并购和全球市场竞争。开放方向由单向学习变成双向互动。

这也带来新的问题：企业出海要面对法律、文化、政治风险；产业升级会触碰既有国际分工；技术追赶越接近前沿，外部限制越强。后发优势在追赶初期最明显，越到高端越需要原创能力和制度信任。

## 十、开放与主权并不矛盾

近代被迫开放造成的历史记忆，使很多人容易把开放与受制于人联系起来。但改革开放的经验表明，主动开放可以成为增强主权能力的手段。通过贸易、学习和竞争，国家获得更多财富、技术和国际影响力，反而能更好地维护自身选择。

关键在于开放的规则由谁制定、风险如何管理、收益如何分配。若开放只让少数地区和群体获益，就会制造内部裂缝；若开放能带动产业升级、教育提升和制度完善，它就会成为现代化的加速器。

## 十一、这一讲与近代史前半段的呼应

晚清面对西方时，常常是被动接受规则；改革开放后的中国，则试图在学习规则的同时参与规则。这个变化背后，是国家统一、政权稳定、工业基础、教育扩展和国际战略空间共同作用的结果。它说明近现代史不是简单从屈辱走向富强，而是从被动卷入世界，逐步走向主动利用世界、塑造世界。
