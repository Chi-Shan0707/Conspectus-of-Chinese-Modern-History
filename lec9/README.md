# 第九讲：土地革命、长征与民族危机的加深

国民革命失败后，中国革命进入一个更艰难的阶段。城市中心道路受挫，国共合作破裂，白色恐怖压缩了公开政治空间；与此同时，日本在东北和华北不断推进侵略，民族危机从边疆压力变成全国性危机。本讲的主线，是中国共产党如何在失败后寻找新的生存与动员方式，以及国内革命如何被外部侵略重新改写。

## 一、1927：从合作破裂到道路重估

1927年的清党与分裂，使共产党失去在国民革命中的合法空间。南昌起义、秋收起义、广州起义等尝试，说明革命力量并没有消失，但也暴露出城市暴动难以持续的问题。工业无产阶级数量有限，城市被国民党政权和地方军警严密控制，单靠夺取中心城市很难打开局面。

农村道路的意义正在这里显现。井冈山根据地不是退却意义上的“躲进山里”，而是在国家权力薄弱、土地矛盾尖锐、地方武装复杂的区域中，重新组织军事、政权和群众。土地革命使贫苦农民从旁观者变成参与者，红军也因此不只是军队，而是与地方社会相互嵌入的政治组织。

## 二、土地革命的双重性

土地革命最直接的功能，是解决农民对土地和生计的诉求。它把“革命”从抽象政治口号转化为村庄中的利益再分配，从而获得真实动员力。但土地革命也有高风险：阶级划分可能扩大社会裂痕，政策过急会破坏生产与基层秩序，军事压力下的根据地治理更容易走向粗糙化。

因此，这一时期的探索不是直线成功，而是在斗争、纠错、再组织中前进。它留下的重要经验，是革命必须回答“谁来参加、为何参加、如何维持组织”三个问题。没有土地问题，就没有农民动员；没有纪律和政权建设，动员又会退化为短期暴力。

## 三、围剿、反围剿与根据地困境

江西中央苏区的发展，一度证明农村根据地具有扩张可能。但国民党多次“围剿”也说明，根据地不能脱离更大的军事和政治环境。第五次反“围剿”的失败，与军事路线、堡垒推进、组织决策都有关系。它不是单纯的战术失误，而是革命组织在高度压力下如何决策的案例。

长征由此发生。它常被写成英雄叙事，但更应先看到它的危机性质：这是在失败中保存核心力量的战略转移。若没有前期根据地积累，长征无从发生；若不能在途中重建领导和方向，长征也可能只是溃散。

## 四、遵义会议的转折

遵义会议的重要性，在于它改变了长征途中的领导与军事指挥格局，也标志着党在极端危机中开始形成更自主的判断。四渡赤水、巧渡金沙江、强渡大渡河、飞夺泸定桥等行动，不能只当作传奇片段，而要看作机动战摆脱被动局面的连续实践。

长征最终保存的力量人数有限，却留下了政治象征、组织骨干和新的根据地落脚点。陕北根据地的存在，使中央红军获得战略支点，也为随后抗日战争中的力量展开提供条件。

## 五、日本军国主义与东北危机

与此同时，日本对华侵略不断升级。1931年九一八事变后，东北沦陷，伪满洲国建立。日本的行动不是偶然边境冲突，而是明治以来大陆政策、资源需求、军部扩张和国际经济危机共同作用的结果。1929年世界经济危机加剧日本国内矛盾，军部更倾向以对外扩张转移压力、获取资源和市场。

华北危机则进一步说明，侵略不再局限于东北。日本通过政治、军事和经济多种方式蚕食中国主权，国民政府“攘外必先安内”的政策越来越难以面对公众民族情绪。民族危机改变了国内政治排序：继续内战的合法性下降，建立抗日民族统一战线的压力上升。

## 六、西安事变与统一战线的形成

1936年西安事变，是国内矛盾与民族矛盾重新排序的标志。张学良、杨虎城发动兵谏，要求停止内战、一致抗日。事件最终和平解决，为第二次国共合作创造了条件。它说明，政治选择并非只由中央政权决定，地方实力派、民众舆论、国际形势和民族危机都会重塑决策空间。

统一战线并不意味着矛盾消失。国共双方仍有战略目标差异，但在日本全面侵华面前，合作成为必要。近现代史中许多转折都带有这种性质：不是理想状态下的共识，而是在危机逼迫下形成的最低共同目标。

## 七、文物流失与国家能力

近代文物流失可作为理解国家衰弱的侧面。战争掠夺、盗掘走私、不平等交易、收藏市场扩张，都使大量文物离开中国。文物不是孤立器物，它连接着主权、文化记忆和国际秩序。一个国家无力保护土地、人民和制度时，也往往无力保护自身文化遗产。

文物回归则不只是情感补偿，也反映国家能力、外交谈判、法律机制和社会财富的变化。它提醒我们，民族复兴并非抽象口号，常常体现在具体而微的修复之中。

## 八、本讲的核心

土地革命回答的是“革命力量如何在失败后重新扎根”，长征回答的是“组织如何在绝境中保存并重建方向”，民族危机回答的是“外部侵略如何改变国内政治优先级”。三条线合在一起，构成1930年代中国历史的底色：革命、战争与国家存亡已经无法分开。

## 九、城市与农村：革命空间的转换

国民革命时期，共产党曾把主要希望放在城市工人运动和国共合作框架内。1927年后，这条路基本被堵死。城市的工会、报刊、学校和公开组织遭到压制，党员和积极分子大量牺牲。革命若继续停留在城市中心，就会不断被强大国家机器围剿。

农村并不是天然革命根据地。它之所以成为可能，是因为国民政府对许多山区、边区、交界地带控制有限，地方社会存在土地矛盾和武装传统，农民对生存改善有强烈需求。毛泽东关于农村包围城市的探索，正是把中国社会结构与革命战略重新对接。它不是照搬俄国十月革命，而是在中国条件下改写革命空间。

## 十、长征的历史记忆与组织意义

长征后来成为重要政治记忆，但不能只停留在精神象征。它实际解决了三个问题：保存一批经过考验的干部和军队，形成新的领导核心，把革命重心转移到更接近抗日前线和西北战略空间的地区。长征的艰苦，使组织内部的纪律、信念和路线问题被极端放大，谁能在危机中提供有效判断，谁就更可能获得领导权威。

遵义会议的意义，也不只是某个人地位上升，而是中国共产党开始在军事和政治路线上摆脱机械指挥，形成更符合中国实际的决策方式。它是失败中的纠错机制，而纠错能力往往比一时胜利更能决定组织生命。

## 十一、民族危机如何重塑革命合法性

九一八事变后，“抗日”逐渐压过单纯的党派斗争，成为全国性政治道德标准。国民政府若继续把主要力量用于“剿共”，就会面对越来越大的舆论压力；共产党提出抗日民族统一战线，也由此获得更广阔的政治空间。

这说明合法性会随历史处境变化。1920年代最能动员青年的可能是反帝反军阀，1930年代最迫切的则是民族存亡。一个政治组织若能把自身目标与时代最紧迫的问题连接起来，就能获得超出组织规模的影响力。
