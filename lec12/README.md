# 第十二讲：改革开放的改革起点

摘要
----
本讲聚焦“改革”的内部动力与主要实践路径：为什么是邓小平推动改革？改革从哪里开始？采取了哪些政策组合（农业包产到户、乡镇企业、地方放权让利、国企改制的起步），以及这些改革带来的短中期效果与长期矛盾。笔记结合课堂讲稿与公开资料，列出时间节点、关键人物、案例与分析。

**核心结论（浓缩）**
- **领导人作用**：改革在很大程度上依赖于以邓小平为代表的强势、务实型领导推动；地方试验（如小岗村）需要中央容忍或支持才能扩展为全国性政策。
- **渐进试错**：从“试点–局部放宽–逐步推广”构成改革常态；中央与地方的互动、财政与行政体制的配套（或缺失）影响改革成效。
- **双轨并进**：农业的家庭联产承包为释放劳动力、提高产出创造条件；乡镇企业和沿海开放（下讲）则把劳动力与资本、海外联系结合，推动非农产业化。

时间线（要点）
----------------
- 1978 年 3rd Plenum（十一届三中全会）确立改革方向，强调拨乱反正与经济建设优先。
- 1978–1979：小岗村等地方试点开始实行包产到户，随后被发现并逐步被上层接受。
- 1980 年代初：农业包产到户迅速推广（1982 年后形成制度化的家庭联产承包责任制），粮食产量显著上升；同时地方开始发展乡镇企业（TVE）。
- 1980s 中后期：国企改革的压力（亏损、效率低下）逐步显现，推动国企改革起步；沿海与外资相关的开放策略随后并行（下一讲主线）。

背景与政治约束
----------------
- 文化大革命后的“拨乱反正”与社会需求：大学恢复招生、科研文化复兴、知识分子和技术人才重新被动员（讲稿中反复提到“教科文领域的拨乱反正”与“释放人的可能性”）。
- 意识形态与现实利益拉锯：改革并非一开始就被普遍接受；存在保守派与谨慎派的反对或顾虑（例如担心破坏集体或国家控制），因此需要中央高层逐步容忍或推动。

主要改革路径（细分）
------------------
1) 农业改革：家庭联产承包责任制（包干到户）
	- 起点：小岗村等草根试验（1978 年），农户把土地分包到户，先试后报。产生了显著产量增长，成为政策扩散的范例。
	- 机制：国家对最低配额（上缴）与超额产出的多轨定价（配额内按计划价，上缴后超额按更高价或市场价），既保障国家供给也激励生产。
	- 时间节点：1979–1983 年从试点扩展到全国，1982 年以后逐步制度化。
	- 影响：粮食产量大幅提升；农村剩余劳动力形成，为后续城镇化和乡镇企业提供人力资源；同时也带来对农村公共投入与长期人力资本投资的结构性影响（研究显示短期收益显著，但对后代教育投资有复杂效应，见参考文献）。

2) 地方工业与乡镇企业（TVEs）
	- 模式差异：苏南模式（以乡镇企业为主、与地方政府嵌入紧密）、温州模式（以民间、小商贩与华侨资本为主，靠市场与草根网络成长）、珠江三角洲/珠江模式（依赖港澳与华侨资本与外向型贸易链）。
	- 动力：包产到户释放的劳动力 + 地方官员的财政激励（地方留成） + “放权让利”的行政政策组合。
	- 问题：财政与税制初期不配套，地方财政与国有体制之间存在张力；TVE 的快速扩张也带来环境与安全问题。

3) 国有企业改革的起步
	- 触发因素：国企效率低、财政补贴压力大；部分国企在竞争中倒闭或被重组，形成改革压力（“倒逼”机制）。
	- 早期路径：先是允许个体经济与集体所有企业成长，缓解就业压力；随后探索放权、承包、企业自主权扩大等改革手段。

4) 财政与行政体制：放权让利
	- 含义：把部分决策权、税收留成与利润分配权下放给地方，以激发地方经济主动性（但也带来“地方保护主义”“竞争式减税”等问题）。

课堂要点摘录（与讲稿对应）
--------------------
- 教师强调：改革并非单一“市场化”魔法，而是由领导人推动+地方试验+制度设计三者共同作用；小岗村如果没有邓小平的态度，改革难以全国推广（讲稿观点）。
- 具体案例如温州、苏南、珠三角被用来说明发展路径的多样性：温州以个体经营见长，苏南以乡镇企业见长，珠三角依赖外部华侨与港澳资本。

深入分析（为什么是邓小平推动？）
--------------------------------
- 个人与制度：邓小平的成功并非只靠个人意志，而是因为他在政治博弈中拥有足够权威与策略灵活性（允许试验、在关键时刻给予保护或放权），形成“中央-地方试错”机制的可持续性。
- 意识形态的调适：邓小平强调务实（“不管白猫黑猫”类的实用主义口吻在后续言论中常被引用），使得意识形态论争被置于次要位置，从而降低改革被否决的风险。
- 结构性契机：文化大革命后的资源错配、粮食与工业产能问题以及国际环境（外资和海外华人资本的可用性）共同形成改革窗口。

案例研究（要点）
-----------------
- 小岗村（Xiaogang）：草根“秘密契约”→产出暴增→被上层发现并被上层采纳，成为农业改革的标志性案例（参见 Xiaogang, Household Responsibility System）。
- 温州：民营/个体经济迅速崛起，靠华侨网络与本地小商品市场，代表草根市场化路径。
- 苏南/乡镇企业：地方与集体资本结合，成为吸收农村剩余劳动力、推动工业化的主力。

改革的短期收益与长期问题
----------------------
- 收益：农业产量与农民收入上升、非农就业扩张、整体经济增长启动与对外贸易扩张（下一讲关注开放）。
- 问题：收入差距拉大（地区与城乡差距）、环境代价、国企改革带来的失业与社会稳定问题、治理与腐败风险、以及对长期公共投资（教育、卫生）的影响存在争论。

课堂讨论建议与问题（可用于复习/答题）
--------------------------------
- 为什么地方试验（如小岗村）能被中央采纳？哪些条件是关键？
- 家庭联产承包制度的激励机制是什么？它为什么提高产出？它的局限在哪里？
- 比较温州与苏南模式，它们为何能成功？成功的制度基础与社会基础有哪些不同？
- 放权让利带来增长动力，但为何同时带来治理挑战？应如何权衡？

----------------

## 深入补充：改革为什么先从农村突破

农村改革率先突破，既因为农村问题最急，也因为改革阻力相对可控。人民公社体制下，平均主义压低农户积极性，农业产出和农民收入长期受限；而包产到户、包干到户能立刻改变收益结构，让“多劳多得”变成可感知的激励。安徽、四川等地的试验说明，农民并不缺勤劳，缺的是把劳动成果与家庭收益连接起来的制度安排。

但家庭联产承包责任制并不是土地私有化。土地集体所有，农户获得承包经营权，这种安排既保留了政治可接受性，也释放了生产积极性。它的巧妙之处在于不先争论抽象姓资姓社，而是用产量、收入和生活改善来证明制度调整的必要。

## 再看邓小平的作用

邓小平的关键，不是亲自设计每个细节，而是在政治上保护试验、承认实践结果，并把发展生产力重新放到中心位置。改革初期许多做法都带有灰色地带，如果没有高层容忍，地方创新很容易被扣上错误帽子。强势领导、地方试验和社会需求三者结合，才形成改革起步的窗口。

改革也从一开始就带着矛盾：放权让利会激励地方，也会造成地方保护；允许市场会提高效率，也会扩大差距；乡镇企业吸收劳动力，也会带来环境和安全问题。改革的历史意义不在于没有问题，而在于它建立了通过继续改革解决问题的路径。

## 九、乡镇企业为何重要

乡镇企业常被低估。它们不是国企，也不是后来意义上完全清晰的私企，而是在集体、地方政府、市场需求和农民创业之间形成的混合组织。苏南模式依托地方政府和集体经济，温州模式依托民间商业网络，珠江三角洲则更早连接港澳资本和外向型订单。不同模式说明，中国改革不是一条路复制全国，而是各地根据资源禀赋寻找出口。

乡镇企业吸收了大量农村剩余劳动力，缓解了城市就业压力，也倒逼国有企业面对竞争。它们把工业化带入县域和乡村，使农民不必完全离土离乡也能进入非农产业。后来中国制造业的广泛基础，与这一阶段的基层工业化密切相关。

## 十、放权让利的两面

放权让利让地方政府和企业有了增长动力。地方能留下更多收益，就愿意招商、办厂、修路、改善营商环境；企业有更多自主权，就更重视成本、销售和利润。但放权也会带来地方保护、重复建设、预算软约束和监管不足。

改革的复杂之处在于，许多政策既是动力来源，也是日后问题来源。不能因为后来出现债务、污染、腐败，就否定早期放权的历史作用；也不能因为早期增长显著，就忽视制度约束的重要。改革史的成熟写法，应当追踪同一机制在不同阶段的不同后果。

## 十一、恢复高考的深层意义

恢复高考常被当作教育事件，其实也是社会流动和国家治理事件。它重新确立了知识、考试和择优录取的重要性，让被“文革”打断的一代人重新看到向上流动的通道。对国家而言，现代化需要工程师、教师、医生、科研人员和管理人才，恢复高考就是重建人才选拔机制。

这件事也改变了改革的社会心理。经历政治运动后，人们需要一种相对可预期的规则来恢复信任。高考并不完美，却比出身、关系和政治标签更能提供公平感。改革开放初期的“释放人的可能性”，很大程度上从这里开始。

## 十二、知青回城与就业压力

知青回城是改革初期必须面对的现实难题。大量青年回到城市，需要工作、住房和生活安排，而计划经济体系无法轻松吸纳。个体经济、服务业、乡镇企业和非国有部门的发展，不只是意识形态松动的结果，也有解决就业压力的现实需要。

这说明改革往往由现实问题倒逼。理论争论可以很久，但吃饭、就业、效率和财政压力每天都在发生。中国改革的务实性，正是在这些具体问题中形成的。

## 十三、价格双轨制的历史位置

改革从计划经济走向市场经济，不可能一夜之间放开所有价格。价格双轨制在一段时期内同时保留计划内价格和市场价格，让企业和地方在完成计划任务后有动力参与市场。这种制度容易滋生寻租和腐败，却也避免了全面价格放开带来的剧烈震荡。

评价双轨制，需要放在转型语境中。它不是理想制度，而是过渡工具。许多改革政策都带有这种特征：不纯粹，不漂亮，却能在政治和社会承受范围内推动变化。

## 十四、改革的思想条件

“实践是检验真理的唯一标准”之所以重要，是因为它打破了僵化教条对政策讨论的束缚。改革并不是先在理论上完全证明，再进入现实；相反，许多突破先由地方试验产生结果，再倒逼理论更新。实践标准把判断重心从抽象口号转向实际效果。

这也是中国改革的一个基本方法：不把意识形态争论推到最前面，而是让事实、产量、收入、就业和生活改善说话。它降低了改革初期的阻力，也使“摸着石头过河”成为一种制度风格。
